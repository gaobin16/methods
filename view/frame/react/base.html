警告：

  • 组件名称总是以大写字母开始。
  • 举例来说, <div /> 代表一个 DOM 标签，而 <Welcome /> 则代表一个组件，并且需要在作用域中有一个 Welcome 组件。

生命周期：

  • componentDidMount() 钩子在组件输出被渲染到 DOM 之后运行。
  • componentWillUnmount() 生命周期方法是组件从 DOM 中移除之后运行

正确地使用 State(状态)

  • 不要直接修改 state(状态)
  • state(状态) 更新可能是异步的
  • state(状态)更新会被合并
    // 正确
    this.setState((state, props) => ({
      counter: state.counter + props.increment
    }));
    // 正确
    this.setState(function(state, props) {
      return {
        counter: state.counter + props.increment
      };
    });
